## Журнал движения нефтепродуктов на АЗС  

### Описание проекта
Настольное Python-приложение для учета поступления и отпуска нефтепродуктов с анализом и визуализацией данных.

## Функционал
**•**  Добавление приёма и отпкска конкретной марки топлива;  
**•** Хранение данных - в CSV;  
**•** Сохранение журнала в excel файл, с датой сохранения в имени файла;  
**•** Графический анализ приёма и отпуска топлива;  
**•** Визуализация данных;  
**•** GUI Tk.

## Запуск

pip install -r requirements.txt
